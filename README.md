# 蕾米球Bot

名为"蕾米球"的QQBot，采用Python编写，基于NoneBot2构建。

蕾米球当前在以下项目的基础上构建：

https://github.com/nonebot/nonebot2

https://github.com/NapNeko/NapCatQQ

## 插件介绍

auto_kusa: 生草系统自动生草

kusa_helper: 生草系统小助手

auto_G: 生草系统银行、测G相关

params: 自己写的各种方便写代码的函数
